GraphAPI Springy
================

This module allows use of the Springy force-directed graph library
(http://getspringy.com/) with Drupal's GraphAPI module.

Installation
============

1. Download the Springy library from http://getspringy.com/.
2. Unzip it, and rename the resulting folder to 'springy'.
3. Move this folder to sites/all/libraries (or a similarly suitable location).
   You should have a file sites/all/libraries/springy/springy.js
